Architecture flow

Where the request goes

Use this as the map before troubleshooting. Click a journey step above, then watch the matching layer highlight here.

baseline
DNS / Route 53 Domain resolves to ALB DNS. Not required for UI, but required for friendly HTTPS hostnames.
ALB + TLS HTTPS terminates at ALB using ACM. Listener rules choose a target group by host/path.
Target Group Only healthy EC2 targets receive traffic. 502/503 often starts here.
Private EC2 App Spring Boot listens on 8080. Security group allows ALB to app, not internet to app.
RDS PostgreSQL Private database receives app-to-db traffic only. No public inbound route.
NAT Gateway Not used for inbound user traffic. Used when private EC2 needs outbound internet for packages or updates.
Response Path App returns to ALB, then ALB returns to browser over the existing client connection.

Production practice

Commands, metrics, and resolution paths

Use these when you are inside an incident: prove impact, locate the failing layer, make the smallest safe fix, then verify recovery.

HTTP and endpoint checks 
curl -i http://<alb-dns>/
curl -w 'status=%{http_code} dns=%{time_namelookup} connect=%{time_connect} ttfb=%{time_starttransfer} total=%{time_total}\n' -o /dev/null -s http://<alb-dns>/
curl -H 'Host: app.example.com' -i http://<alb-dns>/cart
curl -i http://localhost:8080/actuator/health

Missed metric ALB TargetResponseTime

Use with p95/p99. If ALB latency rises, check target app, JVM, DB, or downstream wait.

Missed metric UnHealthyHostCount

If this rises, fix health check path, app boot, port 8080, security group, or bad deploy.

Missed metric CPU, memory, disk

CPU scales ASG easily. Memory and disk need CloudWatch Agent. High disk can break logs or deployments.

Missed metric DB connections and latency

High connections or slow queries often appear as p95/p99 before users see hard 5xx errors.

Resolution pattern Bad deployment

Confirm artifact version, compare logs, rollback to last known good S3 artifact, then redeploy safely.

Resolution pattern Networking break

Check SG, NACL, route table, target health, and VPC flow logs before changing application code.
