This folder will contain Terraform code for AWS infrastructure.

Planned components:

- Terraform backend
- GitHub OIDC IAM role
- VPC
- Public subnets
- Private app subnets
- Private DB subnets
- Internet Gateway
- Optional NAT Gateway
- ALB
- EC2
- RDS
- Security groups
- CloudWatch dashboards and alarms
- VPC Flow Logs
